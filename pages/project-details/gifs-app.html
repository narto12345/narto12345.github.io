¿Qué es?

Esta es una aplicación web que permite buscar y descargar gifs. Todos los datos provienen de la API api.giphy de https://developers.giphy.com/

Tecnologías utilizadas:

  • Angular versión 17
  • API para la búsqueda de los gifs: api.giphy

Funcionamiento:

Esta aplicación es básicamente un cliente web de un servicio que permite consultar un repositorio de gifs de cualquier tema en específico. Esta aplicación tiene la capacidad de almacenar y listar un historial de las búsquedas recientes, volver a realizar la consultas en el historial, especificar la cantidad de gifs que se desea obtener, almacenamiento de Local Storage y descarga de los gifs obtenidos.
