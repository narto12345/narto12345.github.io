Sistema Administrador de Proyectos

Esta aplicación es un sistema administrador de proyectos. Está desarrollador con ASP.NET Core (.NET Versión 7), utilizando la tecnología de Razor Pages. Para la persistencia se utiliza una base de datos MySQL, y se encuentra mapeada con Entity Framework.

¿Qué es?

Funcionamiento

Este sietema permite la creación de proyectos, usuarios y tareas. Los usuarios pueden ser añadidos a los proyectos y asignarles tareas, especificando nombre, descripción, responsable, estado, prioridad y fecha de vencimiento. Es una aplicación web monolítica elaborada con Razor Pages en ASP.NET Core con C#, las vistas fueron creadas con páginas Razor y para comunicación con la base de datos MySQL se utiliza Entity Framework.
